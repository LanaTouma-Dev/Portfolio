# Documents Folder

## Resume/CV

Place your resume PDF file here with the name: `Lana_Touma_Resume.pdf`

The portfolio is configured to link to this file from the About section.

### Creating Your Resume

1. Export your resume as a PDF file
2. Name it `Lana_Touma_Resume.pdf`
3. Place it in this directory (`public/assets/documents/`)
4. The download button in the About section will automatically work

### File Naming Convention

- Use underscores instead of spaces
- Keep the filename professional
- Recommended format: `FirstName_LastName_Resume.pdf`

### Alternative Names

If you prefer a different filename, update the path in:
- `src/app/components/about/about.component.html`
